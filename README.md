# react-native-ta-analytics

Modular TA analytics orchestrator for React Native with pluggable adaptors.

## Installation

```sh
npm install react-native-ta-analytics
```

## Usage

```js
import TaAnalytics from "react-native-ta-analytics";

// ...

const result = await TaAnalytics.multiply(3, 7);
```
